# Afonin Game 

2D игра на Python + Pygame.

##  Запуск

Установи зависимости:

pip install -r requirements.txt

Запусти игру:

python main.py

##  Управление
← → — движение

SPACE / ↑ — прыжок

CTRL — выстрел

R — перезапуск после Game Over

##  Зависимости
Python 3.10+
Pygame

##  Описание

Игрок управляет персонажем, стреляет во врагов и набирает очки за убийства. Враги появляются рядом и преследуют игрока.

##  Структура

main.py — основной файл игры

остольное - изображеия для анимации
